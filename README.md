# Hardware Solution Website

A professional English-language website for hardware + software + mechanical solution services.  
Customers describe their needs → we evaluate → custom quote. No fixed pricing.

## Reference
- Style inspiration: [pcbaforge.com](https://www.pcbaforge.com)

## Services
- ⚙️ Hardware Design (PCB / PCBA)
- 💻 Software Development (Embedded / App / Cloud)
- 🔩 Mechanical / Structural Design
- 🎨 Industrial Design (ID) — outsourced

## Site Structure
| Page | Purpose |
|------|---------|
| `index.html` | Homepage — services overview |
| `cases.html` | Case showcase — photos + descriptions, no pricing |
| `quote.html` | Get a Quote — customer request form |
| `about.html` | About Us |
| `forum.html` | Community Forum — traffic & engagement |
| `admin.html` | Admin Panel — upload cases (photos + text) |

## Tech Stack
- Static HTML / CSS / JS
- GitHub Pages deployment
- Lightweight admin via localStorage + JSON (no server required)
